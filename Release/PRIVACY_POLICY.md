# Privacy Policy

Effective date: 13 July 2026

**Lighthouses: Mystery of the Archipelago** is an offline game developed by Aleksey Karakuts.

## Data collection

Version 1.0 does not collect, transmit, sell or share personal information. The game does not require an account and does not contain advertising, analytics, social, payment or cloud-save SDKs. It keeps anonymous aggregate gameplay and runtime-error counters locally for on-device diagnostics; these counters are never transmitted.

## Local storage

The game stores progress, settings, daily-reward status, lives and anonymous local diagnostic counters only on the player device using Unity PlayerPrefs. Removing the app may remove this data.

## Permissions

The Android build requests vibration (`VIBRATE`) for optional haptic feedback. The app does not request network access (`INTERNET`).

## Children

The game is designed for a general audience and does not knowingly collect information from children.

## Changes and contact

If a later version adds online features, this policy will be updated before that version is released.
